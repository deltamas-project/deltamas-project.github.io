    How to use Purchase Order? #

    ðŸŽ‰ Announcement! Purchase Order is released in version 1.0.128.

    Apa itu Purchase Order? Purchase Order adalah aplikasi berbasis web yang digunakan untuk pembelian barang melalui vendor. Aplikasi ini masih tahap pengembangan yang dimana sewaktu-waktu bisa berubah logika dan algoritmanya.

    Vendors #

    Apa itu Vendors? Vendors merupakan penyedia atau penjual yang tersedia di menu aplikasi Purchase Order. Jika tidak ada Data Vendor, maka Purchase Order tidak bisa digunakan sepenuhnya.

    Features #

    1. Menambahkan Data Vendor dengan field yang telah disediakan seperti :
      • Nama Perusahaan
      • Jenis Bidang Usaha
      • NPWP
      • Alamat
      • Nama Penjual
      • Nomor Telepon
      • Nomor Telepon Kantor
      • Alamat Email
    2. Mengedit Data Vendor yang telah di input sebelumnya.
    3. Menambahkan dan Menghapus Data Vendor Pricelist seperti :
      • Nama Barang
      • Jumlah Barang
      • Satuan Barang
      • Harga Barang
      • Diskon
      • PPN

    Proposal or Purchase Requests #

    Apa itu Proposal atau Purchase Requests? Proposal atau Purchase Requests merupakan pengajuan pembelian barang yang ada di Data Vendor.

    Features #

    1. Melakukan pengajuan pembelian barang melalui Vendor yang tersedia.
    2. Memilih barang yang ada di Data Vendor Pricelist.
    3. Menginput Jumlah Barang, Satuan Barang, menjumlahkan total harga barang, dan lain-lain.

    Purchase Order #

    Apa itu Purchase Order? Purchase Order merupakan sebuah proses lanjutan dari pengajuan proposal atau Purchase Requests.

    Features #

    1. Melakukan proses lanjutan dari pengajuan proposal atau Purchase Request menjadi Purchase Order.
    2. Mengatur estimasi pengiriman, dan status Purchase Order.
