    Apa itu Inventory Management? Inventory Management adalah aplikasi berbasis web yawng digunakan untuk managemen barang inventaris. Aplikasi ini masih tahap pengembangan yang dimana sewaktu-waktu bisa berubah logika dan algoritmanya.

    Inventory #

    Apa itu Inventory? Inventory merupakan menu penyimpanan barang yang tersedia di aplikasi Inventory Management. Jika tidak ada Data Inventaris, maka Inventory Management tidak bisa digunakan sepenuhnya.

    Features #

    1. Menambahkan Data Inventaris dengan field yang telah disediakan seperti :
      • Nama Barang
      • Jenis Barang
      • Status Barang
      • Deskripsi
      • Foto Barang (Akan muncul, Jika barang tersebut terhubung ke Data Pegawai.)
    2. Mencari Data Inventaris seperti : Barang Baru, Bekas atau Rusak.
    3. Menghubungkan Data Inventaris tersebut ke Data Pegawai yang akan terhubung ke aplikasi HRMS. ( Data Pegawai > Pegawai > Detail Informasi Pegawai > Inventaris )
    4. Semua barang yang terhubung ke Data Pegawai akan mendapatkan QR Code dan QR Code tersebut bisa discan untuk melihat detail siapa pemilik barang tersebut.
