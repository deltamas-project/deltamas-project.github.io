    Apa itu Passport? Passport adalah Single Sign On (SSO) yang dibuat untuk jalur masuk ke semua aplikasi yang dibangun diatas framework CodeIgniter. Masih banyak kekurangan untuk Passport ini. Kedepannya, metode ini akan dirubah ke metode JWT.

    Bagaimana metode JWT tersebut?

    Jadi penjelasan simplenya…

    1. Pengguna login ke Passport.
    2. Lalu, Jika login berhasil dan akan mendapatkan JWT yang tersimpan di browser pengguna.
    3. Aplikasi yang berbeda akan mengakses JWT tersebut.
    4. Pengguna tidak memerlukan login ke aplikasi yang berbeda.

    Saat ini, Passport menyimpan semua sesi data user ke database bukan ke browser pengguna.
