Informasi Aplikasi
  • 1. E-Absensi Mobile adalah aplikasi yang digunakan untuk melakukan absensi melalui handphone.
  • 2. Jika kamu melakukan absensi menggunakan aplikasi ini, kamu tidak perlu lagi untuk melakukan absensi menggunakan mesin fingerprint.
  • 3. E-Absensi Mobile dalam proses review oleh pihak Google untuk berada di Google Play Store. Jadi untuk saat ini kita download manual dari website ini.
  • 4. Aplikasi ini memerlukan lokasi dan internet, Jadi pastikan lokasi dan internet anda berada di status aktif ya.
  • 5. Pertanggal 02 Mei 2023, kita sudah mulai untuk Go-Live. Jadi mohon partisipasinya untuk menggunakan aplikasi ini. Terimakasih dari kami, Team IT.
